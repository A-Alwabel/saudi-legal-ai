## 📱 **HOW TO USE**

### **1. Install Dependencies**
```bash
cd client-nextjs
npm install
# or
pnpm install
```

### **2. Configure Environment**
```bash
# Create .env.local file
NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_WS_URL=http://localhost:5000
```

### **3. Run Development Server**
```bash
npm run dev
# or
pnpm dev
```

### **4. Access Application**
```
http://localhost:3000
```
